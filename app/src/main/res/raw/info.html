Propósito:

El objetivo de esta aplicación es servir como una lista de seguimiento de elementos que van a completarse.

Su propósito original era ayudar como una lista de la compra, con la posibilidad de eliminar lo que ya tienes en el carrito, centrándose en los artículos que todavía faltan por encontrar en la tienda.

Como el propósito de esta aplicación podía implementarse de más de una manera, la desarrollé como una lista general de seguimiento de elementos, con una forma sencilla de eliminar los elementos seleccionados.

¿Cómo funciona?

Secciones de la aplicación

Pantalla de listas:

En esta sección encontrarás todas las listas creadas.

Aquí puedes ver todo el contenido relacionado con cada lista.

· Para añadir un elemento, rellena los campos y pulsa el botón "Añadir" para insertar una nueva lista (serás redirigido al contenido vacío de la nueva lista).

· Para acceder a un elemento existente, toca una vez la lista deseada para abrirla.

· Para eliminar un elemento, deslízalo horizontalmente.

· Para editar un elemento, mantén pulsado el elemento y rellena los nuevos campos en la ventana emergente.

Pantalla de lista seleccionada:

Aquí puedes ver todo el contenido relacionado con la lista seleccionada.

· Para añadir un elemento, rellena los campos y pulsa el botón "Añadir" para insertar un nuevo elemento (si no se indica cantidad, se establecerá en 1).

· Para eliminar un elemento, deslízalo horizontalmente.

· Para editar un elemento, mantén pulsado el elemento y rellena los nuevos campos en la ventana emergente.

Pantalla de información:

Una guía sencilla sobre cómo usar la aplicación y cuál es su propósito.

Donde te encuentras ahora mismo.

Barra de menú:

Menú sencillo con dos iconos:

· Icono con una "i" dentro de un círculo: abre la pantalla de información.

· Icono con una flecha hacia abajo dentro de un círculo: abre la pantalla general de listas.
